## Installation Dependencies

```bash
# install ionic on you machine
$ npm install -g @ionic/cli
```

```bash
# download npm dependencies for the project
$ npm install
```

## Running the app

```bash
# development
$ ionic serve

# prepare build for iOS
$ ionic build && npx cap sync && npx cap copy && npx cap open ios
```
